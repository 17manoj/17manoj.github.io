# Manoj Portfolio App

This is a personal portfolio website built using React, Vite, TypeScript, and Tailwind CSS. The portfolio showcases my skills, projects, and experience in AI governance and related fields. It also features an AI agent that can interact with users to provide information about my background and projects.

## Features

- **Hero Section**: Displays my name, professional title, and a brief summary with call-to-action buttons.
- **AI Agent**: An interactive AI assistant that answers questions about my background, explains projects, and describes my skills.
- **Projects Showcase**: A dynamic display of my projects with details such as tech stack, descriptions, and links to GitHub and live demos.
- **Experience Timeline**: A visual representation of my work experience, education, and achievements.
- **Skills Visualization**: An interactive section showcasing my technical skills.

## Technologies Used

- **Frontend**: React, TypeScript, Tailwind CSS
- **State Management**: Context API (if needed)
- **AI Integration**: LangChain for AI agent functionality
- **Deployment**: GitHub Pages

## Getting Started

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/manoj-portfolio-app.git
   cd manoj-portfolio-app
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser** and navigate to `http://localhost:3000` to view the application.

## Deployment

To deploy the application on GitHub Pages, follow these steps:

1. Build the project:
   ```bash
   npm run build
   ```

2. Deploy the `dist` folder to your GitHub Pages branch (usually `gh-pages`).
